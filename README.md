# 🚗 K Car 중고차 시세 분석 대시보드 (API Interception)

유튜브 '잡학다식 개발자' 채널에서 다룬 중고차 시세 동적 크롤링 및 데이터 시각화 프로젝트입니다. 
단순한 웹 스크래핑을 넘어, 최신 웹 생태계의 보안(암호화)을 우회하고 네트워크 통신을 가로채는 현업의 방식을 적용했습니다.

## 💡 이론과 현업의 간극 : 왜 이 방식을 선택했는가?

**[이론적 접근의 한계]**
일반적인 파이썬 웹 크롤링을 배울 때는 `requests`나 `BeautifulSoup`을 이용해 HTML 구조를 분석하거나, `Selenium`으로 화면의 글자를 긁어옵니다. 하지만 케이카(K Car)와 같은 최신 대형 플랫폼은 SPA(Single Page Application) 구조로 되어 있으며, 검색어 등 핵심 Payload를 브라우저 단에서 복잡하게 암호화(enc)하여 서버로 전송합니다. 따라서 단순한 정적 크롤링이나 리플레이 공격으로는 원하는 데이터를 수집할 수 없습니다.

**[현업의 우회 전술: Playwright Network Interception]**
이 프로젝트는 억지로 암호화 로직을 리버스 엔지니어링하는 대신, 가장 효율적인 우회 전술을 사용합니다. `Playwright`를 통해 실제 브라우저 환경을 구동하고, 브라우저가 내부적으로 처리한 뒤 서버와 주고받는 순수한 형태의 API 응답 데이터(JSON)를 중간에서 낚아채는(Intercept) 방식을 구현했습니다. 이를 통해 웹사이트의 디자인(UI/UX)이 변경되더라도 크롤러가 고장 나지 않는 견고함을 확보했습니다.

## 🛠 주요 기능
* **동적 API 가로채기**: Playwright를 이용해 케이카 내부의 `/search/list` 통신을 탐지하고 순수 JSON 데이터를 추출합니다.
* **데이터 정제**: 서버에서 전달하는 축약된 날짜 데이터(예: '202206') 및 암호화된 코드값을 사용자가 읽기 편한 형태(예: '2022년 06월')로 전처리합니다.
* **직관적인 대시보드**: Streamlit을 활용하여 수집된 매물들의 가격을 한눈에 비교할 수 있는 시각화 차트를 제공합니다.
* **데이터 내보내기**: 분석된 결과를 로컬 PC에 엑셀(`*.xlsx`) 파일로 다운로드할 수 있습니다.

## ⚙️ 실행 방법 (의존성 관리)

**[모던 파이썬 생태계의 표준 도입]**
구형 방식인 `pip`와 `requirements.txt`를 사용할 경우 발생하는 패키지 충돌(의존성 지옥)을 방지하기 위해, 본 프로젝트는 최신 파이썬 패키지 관리자인 `uv`와 `pyproject.toml`을 기반으로 완벽하게 격리된 환경에서 동작하도록 설계되었습니다.

**1. uv 및 프로젝트 환경 세팅**
```bash
# uv가 설치되어 있지 않다면 먼저 설치합니다. (Windows 기준)
powershell -c "irm [https://astral.sh/uv/install.ps1](https://astral.sh/uv/install.ps1) | iex"

# 프로젝트 폴더로 이동 후 필수 패키지를 설치합니다. (pyproject.toml 기반 자동 세팅)
uv add streamlit playwright pandas openpyxl

# Playwright용 내부 브라우저 엔진을 설치합니다.
uv run playwright install

uv run streamlit run car_dashboard_app.py

명령어를 실행하면 로컬 웹 브라우저가 열리며 대시보드에 접속할 수 있습니다.
